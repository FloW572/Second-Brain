# Medien für die README-Demo

Hier die Demo-Medien ablegen, dann in der [README](../README.md#demo) die auskommentierten
`![...]`-Zeilen aktivieren (das `#` bzw. die `<!-- -->` entfernen).

Empfohlen:
- **`demo.gif`** — kurzer Telegram-Mitschnitt (~10–20 s): eine Notiz/ein Todo erfassen, dann eine
  Frage stellen und die begründete Antwort zeigen. Aufnahme z.B. mit dem Handy-Screenrecorder,
  als GIF exportieren (Breite ~600–800 px, damit die Datei klein bleibt).
- **`dashboard.png`** — Screenshot des Web-Dashboards (Item-Liste + Projekt-Ansicht).
- optional **`telegram.png`** — statischer Screenshot des Chats, falls kein GIF.

Tipp: Dateien möglichst < 5 MB halten (GitHub rendert sie direkt in der README).
